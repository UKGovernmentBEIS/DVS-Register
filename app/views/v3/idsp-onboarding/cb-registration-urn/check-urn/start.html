Check unique reference number (URN)

Before you start the certification process you need to confirm if the digital identity provider that you will be working with has passed the pre-screening.

If they have passed the pre-sceening, they have been issued a unique reference number (URN) that they have shared with you trough email.

This form allows you to check if the unique reference number exists and is associated with digital identity provider that shared this number with you.

To complete the check you will need

  • Verify if a provider is trust-marked
  • Digital identity providers registered name
  • Unique reference number that they provided
