Check unique reference number (URN)

Before you start the certification process you need to confirm if the digital identity provider that you will be auditing has passed the pre-assessment.

If they have passed the pre-assessment, they have been issued a unique reference number (URN) that they have shared with you through email.

This form allows you to check if the unique reference number exists and is associated with digital identity provider that shared this number with you.

To complete the check you will need to:

  • Ensure you have the digital identity provider's registered name
  • Ensure you have the unique reference number that they provided
