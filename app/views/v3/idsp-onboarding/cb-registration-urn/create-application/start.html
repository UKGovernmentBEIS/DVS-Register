Create application

For a digital identity provider to be published on the DVS register you will need to create an application.

Once you create an application you will be able to track the status of the application in the Case List.

If they have passed the pre-assessment, they have been issued a unique reference number (URN) that they have shared with you through email.

  • Registered name
  • Unique reference number
